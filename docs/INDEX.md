## 🚀 Quick Start Checklist

**New to The Council?**
1. Read [Quality Workflow Guide](./guides/quality-workflow.md) to understand the system
2. Check [Phantom System Guide](./guides/phantom-system.md) for market research capabilities
3. Review [Mining Drill Guide](./guides/mining-drill.md) for pain point extraction
4. Reference [Repository Analysis Report](./reference/REPOSITORY_ANALYSIS.md) for architecture

**Setting up intelligence systems?**
```bash
npm run quality        # Full quality check + improvement
npm run scout          # GitHub market research
npm run scout:report   # Generate intelligence report
npm run improve        # Learn from successful repositories
npm run brief          # Daily intelligence briefing
```
